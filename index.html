Projects

Project 1: Predicting Physical Exercise Adherence in Fitness Apps using Deep Learning Approach

Collaborated with a team of three to predict fourth-month workout sessions using an LSTM deep learning model, classifying users as adherent or non-adherent with K-Means Clustering. Designed a user-friendly UI with Bootstrap, HTML, and CSS to visualize results through graphs and charts, implementing a feedback loop for dynamic user profile updates based on new session data

Project 2: Analysis of COVID-19 Vaccine Sentiments and Opinions on Twitter

Collaborated in a team of three to perform sentiment analysis on a large dataset of COVID-19 vaccine-related tweets using VADER and TextBlob. Employed a data collection strategy that included web scraping and the Twitter API, and implemented Support Vector Machine (SVM) for sentiment analysis, achieving superior results over other algorithms. The project aimed to provide actionable insights for healthcare organizations and policymakers during the COVID-19 vaccination campaign.
